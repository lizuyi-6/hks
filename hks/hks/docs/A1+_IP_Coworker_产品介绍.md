# A1+ IP Coworker 产品介绍

> AI 驱动的知识产权助手 —— 面向中国小微企业创始人

---

## 一、A1+ 是什么

A1+ IP Coworker 是一个 **AI 知识产权辅助工具**，帮助没有专业 IP 背景的创业者和中小企业主，以最低门槛完成知识产权保护。

简单来说，它做三件事：

| 能力 | 说明 |
|------|------|
| **分析** | 了解你的业务，AI 推荐需要哪些知识产权保护 |
| **准备** | 自动生成商标申请书（Word / PDF），管理你的 IP 资产 |
| **引导** | 提供官方提交流程指引，但**不代替你提交** |

### 产品边界

A1+ 只做**辅助准备**，不做**官方申报**。具体来说：

- 提供：文档生成、风险分析、流程指引、资产管理、到期提醒
- 不做：代替用户向 CNIPA（国家知识产权局商标局）提交申请
- 所有 AI 输出附带免责声明：**仅供参考，以官方为准**

## 二、谁能用

| 用户类型 | 场景 |
|---------|------|
| 小微企业创始人 | 有品牌但还没注册商标 |

| 融资中的公司 | 投资人要求做 IP 尽调 |

## 三、核心功能（6 大模块）

### 1. IP 诊断

**一句话**：告诉 AI 你的业务做什么，它告诉你需要保护什么。

- 输入业务描述、行业、企业阶段
- AI 输出：优先保护资产、风险提示、推荐商标类别
- 结果可直接跳转到商标查重

**举个例子**：你输入"跨境电商 SaaS 平台，面向欧美市场"，AI 会建议你先注册第 35 类（广告销售）和第 42 类（软件服务）商标，同时考虑软件著作权。

---

### 2. 商标查重

**一句话**：输入你想注册的商标名，系统告诉你能不能用。

检查结果分三级：

| 颜色 | 含义 | 建议 |
|------|------|------|
| 🟢 绿色 · 可用 | 未发现冲突 | 放心继续申请 |
| 🟡 黄色 · 近似 | 有相似商标 | 看看具体列表再决定 |
| 🔴 红色 · 冲突 | 明显冲突 | 换个名字吧 |

还会给出具体的相似商标列表、相似度评分，以及**备选名称建议**。

---

### 3. 商标申请书生成

**一句话**：一键生成 Word 和 PDF 格式的商标申请文件。

- 基于查重结果自动填入信息
- 生成后提供 DOCX 和 PDF 两个版本下载
- **自动操作**：生成的商标会自动录入资产台账，并设置到期提醒

---

### 4. 提交流程引导

**一句话**：手把手教你如何去官方提交申请。

- 提供编号步骤列表
- 直接链接到 CNIPA 官方入口
- 你需要自己在官方网站完成提交

---

### 5. IP 资产台账

**一句话**：所有知识产权资产集中管理。

- 自动记录：生成申请书后自动创建资产记录
- 手动添加：已有商标、专利也可录入
- 每条资产显示名称、类型、状态、到期时间
- 支持 4 种资产类型：商标、专利、软著、版权

---

### 6. 提醒中心

**一句话**：重要日期自动提醒，不会错过续展和维权时间点。

- 自动创建提醒：资产到期前 90 天、60 天、30 天、7 天
- 支持邮件通知渠道
- 发送失败可以手动重试

---

## 四、使用流程

从注册到完成商标保护，只需 **7 步**：

```
① 注册登录
   ↓
② IP 诊断 ──→ AI 分析你的业务需要什么保护
   ↓
③ 商标查重 ──→ 检查你的商标名能不能用
   ↓
④ 申请书生成 ──→ 一键生成 Word / PDF
   ↓
⑤ 提交引导 ──→ 引导你去 CNIPA 官方提交
   ↓
⑥ 自动入台账 ──→ 系统自动记录你的资产
   ↓
⑦ 提醒任务 ──→ 到期前多轮提醒
```

**模块之间自动衔接**：诊断推荐类别 → 自动带入查重 → 查重结果带入申请书 → 生成后自动入台账 → 自动创建提醒。不需要重复填写信息。

## 五、扩展功能

除了核心的商标流程，A1+ 还提供以下模块：

| 模块 | 功能 | 使用场景 |
|------|------|---------|
| **侵权监控雷达** | 输入商标名或关键词，扫描网络发现可能的侵权行为 | 定期排查品牌是否被侵权 |
| **竞争对手追踪** | 输入竞品公司名，查询其 IP 资产和活跃度 | 了解竞品的商标/专利布局 |
| **合同 IP 条款审查** | 粘贴合同文本，AI 识别知识产权条款风险 | 签合同前快速检查 IP 相关条款 |
| **专利 / 软著辅助** | 描述技术方案，AI 评估应申请专利还是软著 | 不确定技术成果怎么保护 |
| **行业政策摘要** | 输入行业名称，AI 汇总相关政策与合规提醒 | 关注行业 IP 政策变化 |
| **融资 IP 尽调** | 输入目标公司名，分析其 IP 资产、风险和估值因素 | 融资或并购前的 IP 尽调 |

## 七、常见问题

**Q：A1+ 会帮我提交商标申请吗？**
不会。A1+ 只帮你准备文档和了解流程。你需要在 CNIPA 官网自行提交。

**Q：商标查重结果准确吗？**
基于商标数据快照，结果仅供参考。最终以 CNIPA 官方审查为准。

**Q：生成申请书要多久？**
通常几秒钟，生成 Word 和 PDF 两种格式。

**Q：我不懂知识产权，能用吗？**
完全可以。系统用最简单的语言引导你完成每一步。

**Q：需要付费吗？**
当前为开源项目，可自行部署使用。
