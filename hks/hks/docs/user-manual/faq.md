# 常见问题

## A1+ 会帮我提交商标申请吗？

**不会。** A1+ 只帮助你准备文档和了解提交流程。你需要在 CNIPA（国家知识产权局商标局）官方网站上自行完成申请提交。系统会提供提交流程引导和官方入口链接，但不会代替你进行任何官方申报操作。

## 诊断结果准确吗？

诊断结果由 AI 基于你的业务描述和知识库内容生成，仅供参考。AI 会根据关键词识别你的业务类型并推荐相应的保护策略，但最终建议请以专业知识产权代理机构或官方意见为准。

## 我的商标查重结果为什么是「黄色」或「红色」？

- **黄色（近似）**：系统发现了与你商标名称相似的已注册或正在申请的商标。相似度在 65-87% 之间。建议你查看具体的相似项列表，评估是否继续申请或调整名称。
- **红色（冲突）**：系统发现了明显冲突的商标，相似度超过 87%。强烈建议更换商标名称后重新查重。

你可以参考系统提供的备选名称建议。

## 我可以手动添加已有的商标到台账吗？

可以。在「IP 资产台账」页面，使用「新增资产」表单手动录入你已有的知识产权资产。填写资产名称、类型（商标/专利/软著/版权）、注册号（可选）和到期日期（可选）即可。手动添加的资产也会自动创建到期提醒。

## 提醒是怎么触发的？

提醒在以下情况自动创建：
- 生成商标申请书后，系统自动创建 4 个提醒（到期前 90/60/30/7 天）
- 手动添加资产时（如果填写了到期日期），也会自动创建提醒

提醒由后台 Worker 服务处理，通过邮件渠道发送。如果发送失败，你可以在「提醒中心」手动重试。

## 数据安全吗？

- 密码使用 PBKDF2-SHA256 加密存储，不以明文保存
- 登录凭证使用 JWT Token，存储在 httpOnly cookie 中，防止 JavaScript 篡改
- 数据存储在你自己的数据库中，不会发送到第三方

## 「real」和「mock」模式是什么意思？

- **real 模式**：使用真实的数据源和 API。例如商标查重会使用真实的商标数据快照，AI 分析会使用真实的 LLM 引擎。
- **mock 模式**：使用模拟的演示数据，用于产品演示和功能体验。mock 模式下的结果不代表真实情况。

你可以在工作台的「Provider 健康状态」区域看到每个服务当前使用的模式。

## 生成商标申请书需要多长时间？

通常几秒钟即可完成。系统会生成 DOCX 和 PDF 两种格式的文件供你下载。生成完成后，资产会自动录入台账。

## 侵权监控和竞争对手追踪功能如何使用？

- **侵权监控**：在「侵权监控雷达」页面输入商标名称或关键词，系统会扫描网络发现可能的侵权行为并给出风险等级
- **竞争对手追踪**：在「竞争对手追踪」页面输入竞品公司名称，系统会查询该公司的知识产权状况和活跃度

## 合同审查功能能做什么？

在「合同 IP 条款审查」页面粘贴合同文本，系统会自动识别知识产权相关条款，评估风险等级，提示可能缺失的 IP 保护条款，并给出修改建议。
