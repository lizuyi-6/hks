# IP 诊断

## 什么是 IP 诊断

IP 诊断帮助你快速了解你的业务需要哪些知识产权保护。你只需描述你的业务情况，AI 会自动分析并给出保护建议。

## 操作步骤

1. 在左侧导航栏点击「IP 诊断」
2. 填写业务信息表单：
   - **公司名 / 项目名**（可选）：你的公司或项目名称
   - **业务描述**（必填）：描述你的产品、服务、目标客群和商业场景。越详细，诊断结果越准确
   - **所属行业**（可选）：例如「跨境电商」「SaaS」「餐饮」
   - **企业阶段**（可选）：例如「初创」「上线前」「成长期」
3. 点击「生成 IP 保护建议」按钮
4. 等待几秒，系统会显示诊断结果

## 查看诊断结果

诊断结果包含以下信息：

- **总结**：对你业务 IP 保护需求的整体评估
- **优先保护资产**：建议你优先保护哪些类型的知识产权
- **风险提示**：潜在的风险点
- **下一步行动**：建议你具体应该做什么
- **推荐方向**：系统推荐的 IP 保护路径（商标/专利/著作权）
- **推荐商标类别**：如果你需要注册商标，推荐的尼斯分类号

## 下一步

诊断完成后，点击「进入商标流程」按钮，可以直接跳转到商标查重模块继续操作。

## 注意事项

- 诊断结果由 AI 生成，仅供参考，请以官方意见为准
- 业务描述越详细，结果越有价值
- 你可以多次运行诊断，不同的描述可能产生不同的建议
