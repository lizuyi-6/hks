# IP 资产台账

## 什么是资产台账

资产台账集中管理你的所有知识产权资产，包括商标、专利、软件著作权和版权。

资产可以通过两种方式添加：
- **自动添加**：生成商标申请书后，系统会自动创建资产记录
- **手动添加**：你可以手动录入已有的知识产权资产

## 手动添加资产

1. 在左侧导航栏点击「IP 资产台账」
2. 在「新增资产」表单中填写：
   - **资产名称**（必填）：商标名、专利名等
   - **类型**（必填）：选择「商标」「专利」「软著」或「版权」
   - **注册号**（可选）：官方注册号码
   - **到期日期**（可选）：资产的到期日期
3. 点击「添加资产」
4. 手动添加的资产也会自动创建 90/60/30/7 天的到期提醒

## 查看资产列表

「资产列表」区域显示你所有的 IP 资产，每条记录包含：
- 资产名称
- 资产类型（商标/专利/软著/版权）
- 状态（待审核/有效/已过期/已续展）
- 下一个关键节点
- 数据来源标签（显示来自 real 还是 mock 数据）

## 删除资产

如果需要删除某个资产记录：
1. 在资产列表中找到要删除的资产
2. 点击「删除」按钮
3. 注意：删除操作不可撤销，请谨慎操作

## 资产状态说明

| 状态 | 含义 |
|------|------|
| pending（待审核） | 已生成申请书，等待官方审核 |
| active（有效） | 已注册成功，处于有效期内 |
| expired（已过期） | 注册已过期，需要续展 |
| renewed（已续展） | 已完成续展，继续有效 |

## 注意事项

- 建议定期检查资产台账，确保所有资产信息准确
- 关注「下一个关键节点」字段，及时处理待办事项
- 到期提醒可以在「提醒中心」查看
