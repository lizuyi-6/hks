## 一、产品愿景

**一句话定位：**
> A1+ IP Coworker — 你的专属 IP 法务同事，让每个创业者的第一道护城河不再裸奔。

**产品定义：**
面向中国小微创业者和 OPC（一人公司）的 AI 知识产权主动 Agent。通过主动监控、主动催促、替代准备三大能力，替代传统模式下 80% 的人工 IP 管理工作。

**核心价值主张：**
- 从「被动问答」到「主动 Agent」
- 从「一段建议」到「填好的表格、准备好的文件」
- 从「不知道该保护什么」到「AI 主动告诉你该做什么」

## 二、目标用户与痛点

### 2.1 目标用户画像

| 用户类型 | 规模估算 | 核心痛点 |
|---------|---------|---------|
| 小微创业公司（1-10人） | ~1000 万家 | 没预算请法务，IP 意识薄弱 |
| OPC / 自由职业者 | ~5000 万人 | 不知道该保护什么，随时踩坑 |
| 电商 / 短视频创业者 | ~3000 万家 | 品牌被侵权无感知，投诉无门 |
| 海归创业群体 | ~50 万人/年 | 不熟悉国内 IP 申报流程 |

### 2.2 核心痛点分析

**痛点 1：认知门槛高**
- 不知道商标、专利、著作权有什么区别
- 不知道自己的产品该注册哪种 IP
- IP 基础知识几乎为零

**痛点 2：成本高**
- 商标注册代理费：1500-3000 元/件
- 专利申请代理费：5000-20000 元/件
- 专业 IP 顾问：3-10 万元/年
- 对初创公司来说，IP 保护是奢侈品

**痛点 3：流程复杂**
- 商标分类不理解，选错分类 = 保护无效
- 申报表格繁琐，填写规范多，容易被驳回
- 流程节点多（申请→受理→审查→公告→下证），历时 8-14 个月

**痛点 4：被动、滞后**
- 商标过期不知道，被抢注了才反应过来
- 被侵权了无感知，等发现时损失已发生
- 竞争对手布局了什么专利，完全不知道

## 三、核心功能

### 3.1 功能矩阵

| 功能模块 | 功能描述 | 用户价值 |
|---------|---------|---------|
| **IP 诊断** | 输入业务描述，3 分钟 AI 分析该保护什么 | 从 0 到有方向 |
| **商标查重** | 实时检索商标局公开数据，判断冲突风险 | 避免白花钱注册被驳 |
| **申请书生成** | AI 自动生成符合规范的商标/软著申请书 | 省 80% 准备时间 |
| **IP 资产台账** | 统一管理所有商标/专利/著作权 | 告别表格和记忆 |
| **主动续展提醒** | 商标到期前 3 个月自动催促 | 防止过期被抢注 |
| **侵权监控** | 监控全网仿牌、盗图、假冒店铺 | 第一时间发现被侵权 |
| **竞争对手追踪** | 追踪竞争对方新注册商标/专利动态 | 知己知彼，提前布局 |
| **合同审查** | 粘贴合同条款，AI 分析 IP 风险点 | 不花律师费也能审合同 |

### 3.2 核心使用场景

**场景 1：新产品上线前**
> 用户：帮我看看这个产品名能不能用
> Coworker：✅ 名称查重分析 / ⚠️ 有近似商标在注册中 / 建议修改方案已生成

**场景 2：商标注册**
> 用户：帮我注册这个商标
> Coworker：✅ 分类建议已出 / ✅ 申请书已生成 / 确认后提交
> 用户：确认
> Coworker：申请文件已准备好，请在商标局官网核对提交，预计 8-14 个月下证

**场景 3：主动续展提醒**
> Coworker：⚠️ 您的商标「XX」将在 90 天后到期，届时原注册人将失去该商标专用权，是否需要我帮您准备续展文件？

## 四、ROI 测算（核心数据）

### 4.1 用户节省测算

| 场景 | 传统方式成本 | A1+ 成本 | 节省比例 |
|------|------------|---------|---------|
| 商标注册 1 件（代理费） | 1500-3000 元 | 订阅制约 99-299 元/月 | **省 80-90%** |
| 商标注册 1 件（时间成本） | 创始人 4-8 小时 | 10 分钟确认 | **省 95%+** |
| 年度 IP 顾问服务 | 3-10 万元/年 | 订阅制 299 元/月 | **省 97%+** |
| 商标侵权监控（年） | 专业机构 5000+/年 | 包含在订阅内 | **省 90%+** |
| 专利申请文件准备 | 代理费 5000-20000 元 | AI 生成 + 用户自提交 | **省 70%+** |

### 4.2 ROI 计算模型

**单个用户年度 ROI：**
- 传统 IP 管理成本：商标注册 1 件（2000元）+ 顾问基础服务（3万元/年）= **32,000 元/年**
- A1+ 成本：企业版 299 元/月 × 12 = **3,588 元/年**
- **节省：28,412 元/年（省 89%）**

**回本周期：**
第一次使用商标注册（省 1500-3000 元），当月即回本。

**投资回报率（年度）：**
(28,412 - 3,588) / 3,588 = **691% 年度 ROI**

## 六、竞争优势与壁垒

### 6.1 竞品对比

| 维度 | 传统代理 | 律所 SaaS | 豆包/通用 AI | **A1+** |
|------|---------|----------|------------|--------|
| 价格 | 高（1500+/件） | 高（万级/年） | 低 | **99-299/月** |
| 主动性 | 无 | 弱 | 无 | **强（主动提醒）** |
| 交付物 | 代理申报 | 文档模板 | 文字建议 | **可直接用的文件** |
| 工作流 | 全流程人工 | 半自动 | 无 | **AI 跑腿，人只确认** |
| 持续服务 | 续展另收费 | 订阅内 | 单次 | **持续主动监控** |

### 6.2 核心壁垒

**壁垒 1：工作流壁垒**
大模型谁都能接，但没人做 IP 领域的深度工作流。Agent 自动填表 + 文件生成 + 提交引导，这套逻辑需要大量行业 know-how 沉淀。

**壁垒 2：场景壁垒**
垂直 IP 领域的 prompt 工程、申报规范知识库、分类判断逻辑，是需要持续迭代的隐性知识。

**壁垒 3：先发壁垒**
早占领用户心智，早积累行业反馈数据，模型越用越准。
