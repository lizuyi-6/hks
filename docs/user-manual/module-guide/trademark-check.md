# 商标查重

## 什么是商标查重

商标查重帮助你检查想注册的商标名称是否与已注册或正在申请的商标冲突。这是商标注册前的重要步骤。

## 操作步骤

1. 在左侧导航栏点击「商标工作流」进入商标查重页面
2. 填写查重信息：
   - **商标名称**（必填）：你想注册的商标名
   - **业务描述**（必填）：你的业务简介，帮助判断商标类别。如果你之前做了 IP 诊断，这里可能已经填好
   - **申请人名称**（必填）：你的公司名称或个人姓名
   - **申请人类型**：选择「企业」或「个人」
   - **类别**（必填）：商标分类号，用逗号分隔。例如输入 `35,42` 表示第 35 类和第 42 类。IP 诊断推荐的类别会自动填入
3. 点击「执行商标查重」按钮
4. 等待系统分析完成

## 理解查重结果

系统会给出三种风险等级：

### 绿色 — 可用

未发现与你商标名称直接冲突的已注册商标。你可以放心进入下一步申请书生成。

### 黄色 — 近似

存在与你商标名称相似的商标。你需要：
- 查看具体的相似商标列表
- 评估相似度和冲突风险
- 根据建议决定是否继续申请或调整名称

### 红色 — 冲突

存在明显冲突的商标。建议你：
- 查看冲突详情
- 考虑更换商标名称
- 参考系统提供的备选名称建议

## 查重结果详情

- **相似/冲突项列表**：每项显示商标名称、类别、相似度评分、注册状态和备注
- **建议与备选方案**：系统给出的操作建议和替代名称

## 下一步

- 如果结果是绿色或你决定继续申请，点击「进入申请书生成」
- 如果结果是黄色或红色，建议先调整商标名称再重新查重

## 注意事项

- 查重基于商标数据快照，不包含实时数据
- 结果仅供参考，最终以 CNIPA 官方审查为准
- 不同类别的相同商标可能可以共存
