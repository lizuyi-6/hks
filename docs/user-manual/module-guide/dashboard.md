# 工作台

## 什么是工作台

工作台是你的首页和控制中心。登录后自动进入工作台，你可以在这里：

- 查看系统核心指标
- 了解各数据服务是否正常
- 查看最近的 IP 资产
- 查看待处理的提醒任务
- 获取系统推荐的下一步操作

## 核心指标卡片

工作台顶部显示三组核心数据：
- **模块数量**：系统提供的功能模块总数
- **核心流程**：主线流程的步骤数
- **法律边界**：显示「辅助准备」，说明系统只做辅助工作

## Provider 健康状态

「Provider 健康状态」区域显示系统各个数据源的运行状况：
- **Available**（绿色）：该数据服务正常运行
- **Unavailable**（红色）：该数据服务暂时不可用

每个服务旁会显示当前使用的是「real」（真实数据）还是「mock」（演示数据）模式。

## 最近资产

「最近资产」区域显示你最近添加或自动生成的 IP 资产概览。如果还没有资产，系统会提示你先完成申请书生成。

## 提醒队列

「提醒队列」区域显示即将到期的提醒任务概览。如果暂无提醒，系统会显示提示信息。

## 智能建议

工作台会根据你的使用进度给出个性化建议，例如：
- 如果你还没有进行 IP 诊断，会建议你「开始 IP 诊断」
- 如果你已完成诊断但未进行查重，会建议你「开始商标查重」
- 如果你有资产即将到期，会提醒你关注

点击建议中的按钮即可直接跳转到对应模块。
