# 商标申请书生成

## 什么是申请书生成

系统根据你提供的商标信息和查重结果，自动生成 Word（DOCX）和 PDF 格式的商标申请书文档。生成的文档包含申请表格、类别建议、风险备注和提交引导。

## 操作步骤

1. 进入商标申请书生成页面（通常从查重结果页点击「进入申请书生成」跳转）
2. 确认申请信息：
   - **商标名称**：从上一步自动带入
   - **业务描述**：从上一步自动带入
   - **申请人名称**：从上一步自动带入
   - **申请人类型**：从上一步自动带入
   - **类别**：从上一步自动带入
   - **风险等级**：从查重结果自动带入
3. 如有需要，可以修改任何信息
4. 点击「生成 Word / PDF」按钮
5. 等待系统生成文档（通常几秒钟）

## 下载文档

生成完成后，你可以：
- 点击「下载 DOCX」下载 Word 格式文件
- 点击「下载 PDF」下载 PDF 格式文件

## 自动操作

申请书生成完成后，系统会自动执行以下操作：

1. **自动入台账**：将商标信息自动写入 IP 资产台账，状态为「待审核」
2. **自动创建提醒**：在商标到期前 90 天、60 天、30 天和 7 天分别创建提醒任务

系统会显示提示：「申请书生成完成后，系统会自动写入 IP 资产台账并创建提醒任务。」

## 下一步

点击「查看提交引导」了解如何将申请书提交到官方系统。

## 注意事项

- 生成的文档仅供参考，请根据实际情况调整
- 文档中的信息基于你提供的输入，请确保准确
- 生成前请确认商标查重结果为绿色或黄色
