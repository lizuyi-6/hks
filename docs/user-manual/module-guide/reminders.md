# 提醒中心

## 什么是提醒中心

提醒中心显示所有计划中的 IP 资产到期提醒。提醒任务在以下情况自动创建：
- 生成商标申请书后，系统自动创建 4 个提醒
- 手动添加资产时，系统也会自动创建提醒

默认提醒时间表：资产到期前 **90 天、60 天、30 天和 7 天**各发送一次提醒。

## 查看提醒列表

1. 在左侧导航栏点击「提醒中心」
2. 列表显示所有提醒任务，按到期时间排序

每条提醒显示：
- **通知渠道**：EMAIL（邮件）、WECHAT（微信）、WECOM（企业微信）
- **到期时间**：提醒计划发送的时间
- **关联资产**：提醒对应的 IP 资产
- **状态标签**：当前任务状态

## 提醒状态说明

| 状态 | 含义 |
|------|------|
| queued（排队中） | 等待发送 |
| processing（处理中） | 正在发送 |
| sent（已发送） | 已成功发送 |
| failed（失败） | 发送失败，等待重试 |
| dead_letter（死信） | 多次重试失败，不再自动重试 |

## 手动重试

如果某条提醒显示「failed」或「dead_letter」状态，你可以手动触发重新发送：
1. 在提醒列表中找到失败的任务
2. 点击「人工重跑」按钮
3. 系统会重新尝试发送该提醒

## 注意事项

- 提醒依赖邮件服务配置，如果 SMTP 未配置，提醒可能无法发送
- Worker 服务需要运行才能处理提醒任务
- 建议定期检查提醒中心，确保没有遗漏的重要提醒
