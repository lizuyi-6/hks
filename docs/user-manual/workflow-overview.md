# 核心流程总览

## 完整的 IP 保护流程

A1+ 提供了一条完整的知识产权保护主线，共 7 个步骤：

```
注册/登录 → IP 诊断 → 商标查重 → 申请书生成 → 提交引导 → 自动入台账 → 提醒任务
```

### 第 1 步：注册 / 登录

创建账户或登录现有账户，进入工作台。

### 第 2 步：IP 诊断

告诉系统你的业务情况，AI 会分析你需要的知识产权保护类型，并推荐优先保护的资产。

- 输入：业务描述、行业、企业阶段
- 输出：保护建议、风险提示、推荐商标类别

### 第 3 步：商标查重

输入你想注册的商标名称，系统会对比已注册商标数据，给出风险等级。

- **绿色（可用）**：未发现冲突，可以继续申请
- **黄色（近似）**：存在相似商标，需要仔细评估
- **红色（冲突）**：有明显冲突，建议更换名称

### 第 4 步：申请书生成

系统根据你的商标信息和查重结果，自动生成 Word（DOCX）和 PDF 格式的商标申请书文档。你可以直接下载使用。

### 第 5 步：提交引导

系统提供详细的提交流程步骤，并链接到 CNIPA（国家知识产权局商标局）官方入口。你需要自己完成提交。

### 第 6 步：自动入台账

申请书生成完成后，系统会自动将商标信息写入你的 IP 资产台账，无需手动操作。

### 第 7 步：提醒任务

系统自动创建提醒任务，在资产到期前的 90 天、60 天、30 天和 7 天发送提醒通知，确保你不会错过重要的续展或维权时间节点。

## 扩展模块

除了核心流程外，A1+ 还提供以下扩展模块：

| 模块 | 功能 |
|------|------|
| 侵权监控雷达 | 扫描网络，发现可能的商标侵权行为 |
| 竞争对手追踪 | 查询竞争对手的知识产权动态 |
| 合同 IP 条款审查 | AI 分析合同中的知识产权相关条款，提示风险 |
| 专利 / 软著辅助 | 评估技术方案，推荐专利或软件著作权保护策略 |
| 行业政策摘要 | 汇总行业相关的知识产权政策和法规变化 |
| 融资 IP 尽调 | 分析目标公司的知识产权资产和风险 |

---

## 流程之间的衔接

各模块之间自动传递数据，减少重复输入：

- IP 诊断的结果会推荐商标类别，可一键进入商标查重
- 商标查重的结果自动带入申请书生成步骤
- 申请书生成后自动创建资产台账记录
- 资产创建时自动设置多个到期提醒
- 工作台会根据你的进度给出下一步操作建议
