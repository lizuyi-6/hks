# 快速入门

## 欢迎使用 A1+ IP Coworker

A1+ IP Coworker 是你的知识产权（IP）智能助手，帮助你：
- 了解你的业务需要哪些知识产权保护
- 检查商标是否与已注册商标冲突
- 自动生成商标申请书文档
- 管理你的所有知识产权资产
- 在重要日期前收到提醒

**重要提示**：A1+ 只帮助你准备文档和了解流程，**不会代替你向政府系统提交申请**。所有申报操作由你亲自完成。

---

## 注册账户

1. 打开 A1+ 网站，点击「创建账户」
2. 填写以下信息：
   - **姓名**：你的真实姓名或公司名
   - **邮箱**：用于登录的邮箱地址
   - **密码**：设置一个安全的密码
3. 点击「注册并进入工作台」
4. 系统自动登录并跳转到工作台

---

## 登录

1. 打开 A1+ 网站，进入登录页面
2. 输入你的邮箱和密码
3. 点击「登录」
4. 成功后跳转到工作台

---

## 界面导览

### 左侧导航栏

左侧栏显示所有功能模块，点击可切换到对应页面。每个模块旁有一个状态标签：
- **Core**：该模块已完全可用
- **Skeleton**：该模块正在开发中，即将推出

### 主内容区

右侧是当前模块的工作区域，包含表单、结果展示和操作按钮。

### 法律边界提示

系统中会出现深色的法律边界提示栏，提醒你：A1+ 仅提供辅助服务，不代替官方申报。

---

## 建议的起始步骤

如果你是第一次使用，建议按以下顺序操作：

1. 先完成 **IP 诊断**，了解你的业务需要哪些保护
2. 然后进行 **商标查重**，检查你的商标名称是否可用
3. 根据查重结果生成 **商标申请书**
4. 按照系统引导完成 **官方提交**
5. 系统会自动将资产记录到 **资产台账** 并设置 **到期提醒**
